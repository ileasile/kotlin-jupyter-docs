mergeExceptions

inline fun mergeExceptions(actions: ExceptionMerger.() -> Unit)

Runs all ExceptionMerger.catchIndependently blocks, even if there was an exception in one or more of them. If there were exceptions, the first one is thrown, and the others are added as suppressed to it. If there was an exception outside of ExceptionMerger.catchIndependently, it is also added as suppressed to any previous exception, which is thrown.

inline fun <R> tryFinally(action: () -> R, finally: () -> Unit): R
